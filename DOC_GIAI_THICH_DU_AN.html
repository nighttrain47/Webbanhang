Hệ thống E-Commerce toàn diện dành cho cộng đồng đam mê mô hình & đồ chơi sưu tầm. Được xây dựng trên nền tảng MERN Stack với kiến trúc hiện đại, bảo mật và có thể phát triển mở rộng.

⚛️ ReactJS + Vite 🟢 Node.js + Express 🍃 MongoDB Atlas ☁️ Vercel + Render 🔐 JWT + OTP
🏗️ Mục 1

Kiến trúc Hệ thống Tổng thể

💻 Mục 2

Nền tảng Công nghệ (MERN Stack)

Dự án áp dụng MERN Stack – hệ sinh thái JavaScript/TypeScript đồng bộ từ Frontend đến Backend, giúp tiết kiệm chi phí bảo trì và dễ dàng tích hợp các thư viện mới.

⚛️

ReactJS + TypeScript

Single Page Application với Virtual DOM. TypeScript kiểm tra lỗi kiểu ngay trong IDE, tránh bug logic khi build.

⚡

Vite (Build Tool)

Khởi động Dev Server tính bằng mili-giây. Build bundle cực nhanh dựa trên ES Modules gốc của trình duyệt.

🎨

Tailwind CSS

Utility-first CSS Framework. Class có sẵn đắp thẳng vào HTML, dựng giao diện Responsive siêu nhanh.

🟢

Node.js

Runtime JavaScript bên ngoài trình duyệt. Kiến trúc Event-Driven phục vụ nhiều request đồng thời mà không bị nghẽn.

🚀

ExpressJS

Web Framework nhỏ gọn thiết lập RESTful API Endpoints và quản lý chuỗi Middleware kiểm soát quyền.

🍃

MongoDB + Mongoose

NoSQL Document Database linh hoạt. Mongoose ODM hỗ trợ Aggregation Pipeline mạnh để phân tích dữ liệu đơn hàng.

💡 Tại sao MERN?
Toàn bộ dự án chỉ sử dụng một ngôn ngữ duy nhất — JavaScript/TypeScript — từ giao diện người dùng đến máy chủ và cơ sở dữ liệu. Điều này giúp giảm thiểu đáng kể thời gian chuyển đổi ngữ cảnh (context switching) và làm tăng tốc quá trình phát triển.
🔐 Mục 3

Luồng Xác thực & Phân quyền

Hệ thống sử dụng hai lớp bảo mật: OTP qua Email để xác minh danh tính khi đăng ký, và JWT Token để duy trì phiên đăng nhập an toàn.

🔑 JWT Token
Hoạt động như "chìa khóa điện tử" chứa thông tin role (Customer/Admin). Middleware verifyToken chặn đứng mọi request không hợp lệ với status 401/403.
📧 OTP (Nodemailer)
Mã OTP có thời hạn 5 phút, gửi qua SMTP thực. Admin sử dụng Token riêng biệt, tách hoàn toàn khỏi luồng Customer, đảm bảo không thể giả mạo quyền.
🛒 Mục 4

Kiến trúc Luồng Thanh Toán (Checkout)

Luồng thanh toán được thiết kế để gom đủ dữ liệu từ nhiều nguồn (giỏ hàng, địa chỉ, phương thức thanh toán) và xử lý an toàn, đồng thời kích hoạt thông báo Email tự động.

⭐ Mục 4.1

Hệ thống Hạng Thành Viên (Loyalty Tiers)

Tự động phân hạng thành viên dựa vào tổng chi tiêu lịch sử, sử dụng MongoDB Aggregation Pipeline để tính toán và cập nhật realtime.

🥉
Cấp 1 – Thành Viên

Chi tiêu < 5.000.000₫
Chiết khấu 2%

🥈
Cấp 2 – Bạch Kim

Chi tiêu 5tr – 20tr ₫
Chiết khấu 5%

👑
Đặc Cấp – VIP

Chi tiêu > 20.000.000₫
Chiết khấu 8%

🗄️ Cơ chế xử lý: Backend sử dụng Mongoose Aggregation Pipeline với $lookup (nối bảng Orders với Customers), $group (tính tổng chi tiêu) và $match (lọc theo mốc tiền) để tự động cập nhật hạng mỗi khi có đơn hàng mới.

Vercel

Host Frontend ReactJS. Global CDN cực mạnh giúp assets UI được phân phối đến người dùng với độ trễ thấp nhất.

🌐

Render

Host Backend Node.js/Express. Tự động scale, SSL miễn phí, tích hợp CI/CD từ GitHub repo.

🍃

MongoDB Atlas

Cloud Database managed service. Backup tự động, replica set sẵn sàng cao, dashboard monitoring.

🤖

UptimeRobot

Giám sát và ping định kỳ 5 phút/lần, "đánh thức" server Render khỏi ngủ đông, đảm bảo API luôn hoạt động.

💡 Vì sao combo này?
Render gói Free sẽ cho server "ngủ" sau 15 phút không có request. UptimeRobot cứ 5 phút gọi GET /api/health — đủ để giữ server tỉnh táo 24/7, đảm bảo trải nghiệm khách hàng không bị gián đoạn hoàn toàn miễn phí.
