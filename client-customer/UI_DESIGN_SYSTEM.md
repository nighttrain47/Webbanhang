# HobbyShop UI Design System
> Inspired by CyStore (https://cystore.com/)

## Colors
| Token | Value | Usage |
|-------|-------|-------|
| Primary | `#1a1a1a` | Header bg, primary buttons, headings |
| Accent | `#FF6B00` | Prices, badges, CTAs |
| Background | `#FFFFFF` | Main content |
| Background Alt | `#F8F8F8` | Sections, newsletter |
| Card Background | `#F5F5F5` | Product image containers |
| Text | `#1a1a1a` | Body text |
| Text Secondary | `#888888` | Meta, captions |
| Text Muted | `#aaaaaa` | Manufacturers, subtle text |
| Border | `#E8E8E8` | Dividers |
| Success | `#22C55E` | In-stock badges |
| Danger | `#EF4444` | Out-of-stock, errors |
| Preorder | `#8B5CF6` | Pre-order badges |

## Typography
- Font: `Inter` (Google Fonts)
- Weights: 400, 500, 600, 700, 800

## Components
- **Buttons**: Pill-shaped (border-radius: 9999px), primary = black, accent = orange
- **Product Cards**: Borderless, whitespace-separated, no "add to cart" on card
- **Badges**: Pill shape, uppercase, letter-spacing
- **Header**: Dark bg, centered pill search, nav bar below
- **Footer**: Dark bg matching header
